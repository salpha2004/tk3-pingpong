How to build:
Android:
run "gradlew assembleDebug" in the project folder.
But we recommend to import the project into Android Studio.

Desktop:
run "ant" in the project folder.
Please make sure to have set JAVA_HOME to the JDK root directory.

Notes:
In the game, the first player to login should click on "Create Game" first.
